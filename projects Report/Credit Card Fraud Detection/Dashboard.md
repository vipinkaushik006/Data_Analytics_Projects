# Credit Card Fraud Detection Dashboard

## 📌 Project Overview
The **Credit Card Fraud Detection Dashboard** is a data analytics project built using **Power BI** to analyze and visualize fraudulent credit card transactions.

This dashboard helps identify **fraud patterns based on city, merchants, card types, and transaction timings**. The goal of this project is to provide insights into fraudulent activities and help understand how fraud transactions are distributed across different locations and merchants.

The project demonstrates how **data visualization and business intelligence tools** can help detect suspicious transaction trends and improve fraud monitoring.

---

# 🎯 Project Objectives
- Identify **fraudulent transactions across different cities**
- Analyze **fraud distribution by card type**
- Detect **high-risk merchants involved in fraud**
- Understand **fraud patterns based on transaction time**
- Compare **fraudulent transactions with normal transactions**
- Build an **interactive fraud monitoring dashboard**

---

# 📈 Dashboard Insights

## 1️⃣ Fraud by City
The dashboard highlights fraud cases occurring across different cities.

Cities analyzed include:

- Chennai
- Kolkata
- Mumbai
- Hyderabad
- Pune
- Bangalore
- Delhi

**Key Insight:**  
Chennai shows the highest number of fraud cases, followed by Kolkata and Mumbai.

---

## 2️⃣ Fraud by Card Type
The dashboard analyzes fraud cases across different credit card types.

Card types include:

- Amex
- MasterCard
- Visa
- RuPay

**Key Insight:**  
Amex cards show the highest number of fraud cases compared to other card types.

---

## 3️⃣ Fraud vs Normal Transactions
The dashboard compares **fraudulent and legitimate transactions**.

- Fraud Transactions: 46  
- Normal Transactions: 554  

**Key Insight:**  
Fraud transactions represent a **small percentage of total transactions**, but identifying them is crucial for financial security.

---

## 4️⃣ Fraud by Merchants
The dashboard identifies merchants where fraud transactions are more frequent.

Merchants analyzed include:

- Myntra
- Amazon
- Uber
- Zomato
- Flipkart
- Swiggy
- Reliance

**Key Insight:**  
Certain merchants show relatively higher fraud activity, which may require additional monitoring.

---

## 5️⃣ Transaction by Hour
This visualization shows the **hour when fraudulent transactions occur**.

**Key Insight:**
